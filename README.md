## 📁 Projects

### 1. 🎬 Netflix Content Analysis (`netflix_analysis.ipynb`)

**Goal:** Explore Netflix's content library to uncover trends in genres, release patterns, and global content distribution.

**Key Findings:**
- 📈 Netflix content additions grew by **340%** from 2015 to 2020
- 🇺🇸 USA, India, and UK account for **60%** of all content  
- 🎭 Drama is the most common genre (35%), followed by Comedy (20%)
- 📺 TV Shows average slightly higher ratings than Movies

**Techniques Used:**
- Data cleaning and missing value imputation
- Groupby aggregations and time-series analysis
- Geographic distribution analysis
- Bar charts, pie charts, heatmaps with Seaborn/Matplotlib

**Tech Stack:** Python · Pandas · Matplotlib · Seaborn · Plotly

---

### 2. 🏠 House Price Prediction (`House_Price_Prediction.ipynb`)

**Goal:** Build a regression model to predict residential house prices based on property features.

**Results:**
- 🎯 Best model R² Score: **0.89**
- 🏆 Top features: Overall Quality, Ground Living Area, Garage Cars

**ML Pipeline:**
1. EDA and outlier detection
2. Feature engineering (log transforms, interaction terms)
3. Missing value imputation
4. Encoding categorical features
5. Model comparison: Linear Regression, Ridge, Lasso, Random Forest
6. Hyperparameter tuning with GridSearchCV

**Tech Stack:** Python · Pandas · Scikit-learn · Matplotlib · Seaborn
